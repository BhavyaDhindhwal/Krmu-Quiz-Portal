# KRMU Quiz Portal

A Full Stack Quiz Management System developed using Node.js, Express.js, MongoDB, HTML, CSS and JavaScript.

## Features

- Student Panel
- Admin Panel
- Authentication
- Subject Management
- Question Management
- Quiz System
- Result Management
- Certificate Generation
- PDF Download
- Dashboard Analytics

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB

## Installation

### Backend

cd backend

npm install

npm start

### Frontend

Open frontend/index.html
